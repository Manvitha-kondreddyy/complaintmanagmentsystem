# Public Complaint Management System

A Java-based desktop application where citizens can report civic issues and officials can resolve them.

## Features
- 3 Role-based access: Citizen, Official, Admin
- Submit and track complaints
- Update complaint status
- Admin dashboard with filters
- GUI built with Java Swing
- Cloud MySQL database (Railway)

## Technologies Used
- Java 21
- Java Swing (GUI)
- JDBC
- MySQL (Railway cloud)
- IntelliJ IDEA
- Git & GitHub

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Add MySQL connector JAR to libraries
4. Run LoginScreen.java
